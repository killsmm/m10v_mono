[概要]
 SD1 Card側にSDカードを差し込み、プログラムを実行すると
 SDカードをクイックフォーマットします。


[コマンド例]
 cui_sample.sh media_format


[制約事項]
 このサンプルは、下記以外の操作には対応していません。

[処理の流れ]
 1. IPCU初期化

 2. Boot-up

 3. Media Format

 4. IPCU初期化解除

 5. 終了


[詳細説明]
1. IPCU初期化
 コマンド送受信とイベント受信用のメールボックスを開通して、
 各コールバック関数を登録します。

2. Boot-up
 RTOS側へBootしたことをコマンド送信して通知します。

3. Media Format
 RTOS側へFormatのコマンド送信します。

4. IPCU初期化解除
 コマンド送受信とイベント受信用のメールボックス開通を解除します。

5. 終了
 電源offかハードウェアリセットをしてください。
