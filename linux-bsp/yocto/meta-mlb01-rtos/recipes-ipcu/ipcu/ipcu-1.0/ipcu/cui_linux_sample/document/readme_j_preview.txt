[概要]
 Previewを表示します。


[コマンド例]
 cui_sample.sh preview


[処理の流れ]
 1. IPCU初期化

 2. Boot-up

 3. Preview

 4. 画像処理パラメータ変更

 5. IPCU初期化解除

 6. 終了


[詳細説明]
1. IPCU初期化
 コマンド送受信とイベント受信用のメールボックスを開通して、
 各コールバック関数を登録します。

2. Boot-up
 RTOS側へBootしたことをコマンド送信して通知します。

3. Preview
 RTOS側へモード変更(IDLE->Preview)のコマンド送信します。

4. 画像処理パラメータ変更 
 コンソールから画像処理パラメータ入力待ちします。
 入力された画像処理パラメータ内容でコマンド送信します。

5. IPCU初期化解除
 コマンド送受信とイベント受信用のメールボックス開通を解除します。
